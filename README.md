<p align="justify">Repositório dedicado ao estudo de CSS, desenvolvendo um site responsivo baseado na página da <a href="https://universe.leagueoflegends.com/en_US/region/shadow-isles/">Ilha das Sombras</a> de League of Legends. Este estudo à parte foi ministrado a princípio pelo canal <a href="https://www.youtube.com/channel/UCkjoHfkLEy7ZT4bA2myJ8xA">Easy Tutorials</a>, ao qual mudei de acordo com minhas vontades. O principal intuito com este projeto foi transparecer minha evolução com estudo de CSS, sendo observado nos efeitos de imagens, gradients e estilo da página. Com isto, a primeira parte deste estudo está devidamente concluída.

## 🍃HTML
- *HTML serve como um escopo, uma linguagem de marcação para direcionar-nos quando formos modificar os elementos contidos nesta com CSS e Javascript. Esta, principalmente, fora usada para criação de listas e sections, além de div para posicionamento e reforço na estilização de imagens e textos.*

## 🍃CSS
- *O CSS ajuda-nos a estilizar a página marcada pelo CSS. Com esta tecnologia, páginas simples apenas com o escopo marcado transformam-se em layouts que facilitam e aprimoram o visual do site. Neste projeto em específico, podemos observar o uso de um menu responsivo, overlay, gradients, keyframes e transform scale, além da responsividade.*</p>

## 🍃JAVASCRIPT
- *O Javascript fora utilizado sobre a barra de navegação lateral. Assim como observado, a lógica do código demonstrou como esconder e mostrar o navbar através do clique na imagem lateral.*
